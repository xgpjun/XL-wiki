# 插件命令

xgplottery2 / xl

子命令与主要负责功能有 ：

- convert  数据迁移，需要同时安装两版插件。
- count    设置玩家抽奖次数
- crate     抽奖箱
- cumulatedReward   累计抽奖
- draw    命令抽奖
- give     给予玩家实体钥匙
- help    帮助信息

- key     设置玩家虚拟钥匙数量
- manage   管理界面（功能很少）
- particle   粒子特效
- reload   重载
- preview  奖池预览

更详细的介绍请在游戏内查看。
